Spring Security & XSUAA
  • GET /v1/sayHello request that provides XSUAA user token details, but only if token matches.
  • GET /v1/method request executes a method secured with Spring Global Method Security.
  • GET /v1/getAdminData request to read sensitive data via Global Method Security. You will get a 403 (UNAUTHORIZED), in case you do not have Admin scope.
  • GET /v2/sayHello request that logs generic JWT info, but only if token matches.
  • GET /v3/requestClientCredentialsToken request that requests the client credentials JWT token and writes it into the log.
  • GET /v3/requestJwtBearerToken request that exchanges a JWT token for a potential different client.
